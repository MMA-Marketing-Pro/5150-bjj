## Content Outline (curation pages)

1. **Hero** — H1 like "The 5 Best BJJ Gyms in Rockland County (Including 5150)"
2. **Lead** — honest framing. Something like: "Ranked by quality of instruction, student reviews, class variety, and community. 5150 is #1 because of X, Y, Z — but here's the full list."
3. **The list** — 5 real academies in Rockland County (5150 + 4 real competitors). For each: name, address, what they're best at, star rating summary, notable feature.
4. **Why 5150 tops the list** — one paragraph with real differentiators (Gracie lineage, family-first, Tiny Sharks program)
5. **Decision framework** — "Pick 5150 if ... Pick X if ... Pick Y if ..." — honest advice
6. **CTA** — "Book a free class at 5150"
7. **Schema** — `ItemList` + `BreadcrumbList` + `FAQPage`

**Note on honesty:** these pages work because they're not puff pieces. Name real competitors, be fair, but make 5150's specific strengths clear. Search engines (and readers) reward this.

## Content Outline (comparison pages)

1. **Hero** — H1 like "BJJ vs Wrestling: Which Should You Train?"
2. **TL;DR table** — 5 rows: cost, fitness value, self-defense, competition scene, kids-friendliness. Side-by-side comparison.
3. **Deep comparison** — 400–600 words of real analysis, written with the voice of an experienced grappler
4. **When to pick each** — decision framework
5. **How 5150 fits** — "If you chose BJJ, here's where to start in Rockland County"
6. **FAQ** — 5 questions
7. **Schema:** `Article` + `FAQPage` + `BreadcrumbList`
